Getting started
---------------

Be sure you have Node.js, npm and npm-run installed.
Tested with node 8.4.0, npm 5.3.0 and npm-run 4.1.2.

#### Install npm packages

```bash
npm install
```

#### Run project

Run the gulp script in your project directory:

```bash
npm run gulp
```
